\## Описание



Веб-приложение на ASP.NET Core Web API, реализующее портал для управления научными публикациями факультета.



\### Основной функционал:

\- \*\*RESTful Web API\*\* для сущности "Публикации" со всеми CRUD-операциями.

\- \*\*Клиентское HTML-приложение\*\* на чистом JavaScript для взаимодействия с API.

\- \*\*Пагинация\*\* для удобного отображения больших объемов данных.

\- \*\*Серверная часть:\*\* ASP.NET Core 8, Entity Framework Core (Code-First), MS SQL Server.

\- \*\*Тестирование:\*\* Полное покрытие основного контроллера модульными тестами с использованием xUnit и InMemory Database.

\- \*\*CI/CD:\*\* Настроен непрерывный процесс интеграции с помощью GitHub Actions для автоматической сборки и прогона тестов при каждом push в репозиторий.
